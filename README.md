# Eye Care and Vision Working Group

The Eye Care and Vision Working Group for the OHDSI community was formed in 2022, comprised of imaging research scientists and observational health researchers familiar with OMOP CDM. The working group aims to progress standardization of ophthalmology data into the OMOP CDM model.

## Links

* Website: [https://ohdsi.github.io/EyeCareVisionWG/](https://ohdsi.github.io/EyeCareVisionWG/)
* GitHub: [https://github.com/OHDSI/EyeCareVisionWG](https://github.com/OHDSI/EyeCareVisionWG) 

## How to Contribute
We welcome contributions from the Eye Care & Vision community! You do not need special permissions to get started — anyone with a GitHub account can contribute.

**Ways to Contribute**

You can contribute in many ways, including:

  - Reporting bugs or data issues
  - Proposing enhancements or new ideas
  - Improving documentation or examples
  - Contributing code, notebooks, or scripts
  - Sharing use cases, pilot datasets, or implementation feedback

**Getting Started**

1. Create a GitHub account (if you don’t already have one).
2. Explore the repository and review the README and any existing documentation.
3. Open an Issue to ask a question, suggest an idea, or flag an issue — this is a great first step.
4. Submit a Pull Request (PR) if you’d like to propose a change:
   ...*Fork the repository
   ...*Make your changes in a branch
   ...*Open a pull request with a brief explanation of what you’ve changed and why


**Contribution Guidelines**

- Please keep contributions focused and well‑described.
- Be respectful and constructive — this is a collaborative, community‑driven project.
- If you’re unsure where to start or whether an idea fits, opening an Issue for discussion is encouraged.

For a step‑by‑step overview of contributing to open‑source projects on GitHub, see GitHub’s documentation:
https://docs.github.com/en/get-started/exploring-projects-on-github/contributing-to-open-source

Thank you for helping advance interoperable, reusable eye care and vision research resources!
