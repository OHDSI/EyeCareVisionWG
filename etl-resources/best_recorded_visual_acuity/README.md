# ETL Conventions for Best Recorded Visual Acuity (BRVA)
Visual acuity (VA) is a critical measure for assessing outcomes in ophthalmology. In early 2025, the Eye Care and Vision Research Workgroup, specifically the VA OMOP subgroup, came to consensus on the extract-transform-load (ETL) process for identifying a single BRVA (for the right, left, and both eyes) measurement from the multiple VA measurements that might exist in a given source database (e.g., electronic health record (EHR))
